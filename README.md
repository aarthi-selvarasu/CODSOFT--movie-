# Movie Rating Prediction

This project predicts movie ratings using machine learning.

## Description
The project uses historical movie data such as duration, genre, director, actors, and votes to predict movie ratings.

## Technologies Used
- Python
- Pandas
- Scikit-learn

## How to Run
1. Install required libraries
2. Run the Python file

## Output
The model displays movie details and predicts the rating for a selected movie.
